# tram_history.qml

Layer struscture for maps of historical tram or other mass transit lines

## Layer structure

### lines

1. start_date
2. end_date
3. non_revenue
4. note

### terminals

1. name
2. name_int
3. start_date
4. end_date
5. note

### notes 

1. name
2. name_int

### streets

separate otional layer, just copy from osm (overpass-api). 

Required fields are

1. name
